# StaRPG


## Description

StaRPG is a Python project that aims to create a graphics library based on the PyGame engine but more accessible while maintaining maximum flexibility

## Featuring (eventually)
- Map creation & gestion
- Spritesheet gestion
- Animations
- Menu & Popup
- Triggers / Actions
